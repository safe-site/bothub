How to Use TeleBots!

How to Use

Welcome to the TeleBots tool! This tool allows you to send SMS messages to multiple Telegram users using different bots.

Step 1: Choose a Bot

Select a bot from the dropdown menu. You can choose from predefined bots or create your own bot and obtain the API key.

Below are the steps to follow to generate your own token.

  1. Search BotFather on Telegram.
  2. Type /start to get started.
  3. Type /newbot to get a bot.
  4. Enter your Bot name and unique Username, which should end with the bot.
  5. Then, you would get your Bot token.

Step 2: Enter Chat IDs

Enter the Telegram Chat IDs of the users you want to send messages to. Separate multiple Chat IDs with commas.

How to find your User ID in Telegram?

  1. Open your Telegram app.
  2. Type “userinfobot” in your contacts search bar.
  3. If you can't find this “profile,” try adding an “@” in front of the search query.
  4. When you've located @userinfobot, click or tap it.
  5. Then, select Start at the bottom of the chat.

Step 3: Enter Message

Write your SMS content in the textarea provided. You can also attach an image if needed.

Step 4: Send Message

Click the "Send Message" button to send your SMS to the specified users. The tool will display the overall count of successful and unsuccessful messages.

Important Note: Ensure that you have started the bot on Telegram before attempting to send messages. If the user has not started the bot, messages cannot be sent to them.

View Results

Click on the overall count to view details about successful and unsuccessful Chat IDs.
